# AutoHotkey Shortcuts

This repository contains a small AutoHotkey script to enhance mouse behavior:

## Features

- **Quick Copy** — hold left mouse button 0.4s to send `Ctrl+C`
- **Quick Paste** — hold right mouse button 0.4s to send `Ctrl+V`
- **Middle Button** — press middle mouse button to send `Alt+Left`

## Usage

1. Install AutoHotkey from https://www.autohotkey.com
2. Run `main.ahk`
